# xyz_gen

`xyz_gen` is a Dart package designed to generate boilerplate code, reducing the amount of repetitive code you need to write.

## Getting Started

### Installation

Add this to your package's `pubspec.yaml` file:

```yaml
dependencies:
  xyz_gen: ^0.19.0
```
